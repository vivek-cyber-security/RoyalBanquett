# 🍽️ Royal Banquett

A modern and responsive web application designed to showcase and manage banquet services, bookings, and event details. This project provides an elegant interface for users to explore services and make inquiries easily.

---

## 🚀 Features

* 🏛️ Elegant and responsive UI design.
* 📅 Event / Banquet booking interface
* 🖼️ Image gallery for showcasing venue
* 📞 Contact / Inquiry form
* 📱 Mobile-friendly design
